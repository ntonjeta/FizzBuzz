# FizzBuzz

Write a programm that prints the numbers from 1 to 100. But:

- for multiples of three print “Fizz”
- for multiples of five print "Buzz"
- for multiples of both print "FizzBuzz" 

Sample output looks like below:
```
1
2
Fizz 
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
16
17
Fizz
19
Buzz
... etc up to 100 
```
